# Capstone AWS DevOps Secure

## Project Title

Building a Secure, Automated, and Monitored AWS Infrastructure Using DevOps, DevSecOps, and AI Practices

## Project Overview

This capstone project implements a secure, automated, and monitored AWS infrastructure delivery workflow using Git, Jenkins, Terraform, DevSecOps tooling, AI-assisted improvements, Prometheus, and Grafana.

The project is designed to solve common infrastructure challenges such as manual deployments, inconsistent environments, weak security controls, limited observability, and lack of automation.

## Task 1: CI/CD Foundation

Task 1 establishes the version control and CI/CD foundation of the project.

### Tools Used

- Git
- GitHub
- Jenkins
- VS Code
- Ubuntu/WSL

### Task 1 Implementation

The first phase includes:

- Git repository setup
- GitHub remote repository integration
- Branching strategy
- Jenkins installation
- Jenkins plugin configuration
- Jenkins credentials setup
- Jenkins pipeline creation
- Automated source checkout and validation
